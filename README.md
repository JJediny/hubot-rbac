# hubot-rbac

RBAC authorization framework for hubot

See [`src/rbac.coffee`](src/rbac.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-rbac --save`

Then add **hubot-rbac** to your `external-scripts.json`:

```json
[
  "hubot-rbac"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
